# 二叉搜索树

BST 的**中序遍历**结果是有序的（升序）。

- 左、右子树遍历**升序** 
- 右、左子树遍历**降序** 

## 操作

BST 相关问题的代码逻辑：

```java
public void bst(TreeNode root, int target) {
    if (root.val == target) {
        // TODO 操作
    }
    bst(root.left, target);
    bst(root.right, target);
}
```

常见操作

1. 判断 BST 的合法性
2. 在 BST 中搜索元素
3. 在 BST 中插入一个数
4. 在 BST 中删除一个数
