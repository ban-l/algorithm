# 字符串(字符串+数字)排序

## 数据排序

假设有如下字符串"A12"，其中"A"表示数据类型(A-Z)，"12"表示数据序号(0-9)。

现在需要对一组数据先按照数据序号，再按照数据类型 进行排序。

例如：

```text
["B3","D2","F1","A9","D12","A2","C1","Z0","B1"] => ["Z0","B1","C1","F1","A2","D2","B3","A9","D12"]
```

## 思路(根据字符串得出一个值，比较值大小，再还原)

根据字符的ascii码，把字符串数组转成10进制数组，使用十进制比较大小，再还原回字符串数组。

**A的ascii码是65，Z的ascii码是90，所以使用91进制**

1. 转化：91进制 转 10进制
2. 比较：10进制比大小
3. 还原：10进制 转 91进制

## 代码

```java
public class SortString {
    
    public String[] sort(String[] s) {
        int n = s.length;
        int[] nums = new int[n];
        // 1. 91进制 转 10进制
        for (int i = 0; i < n; i++) {
            String temp = s[i];
            // char转int
            int i1 = temp.charAt(0) - '0';
            int i2 = Integer.parseInt(temp.substring(1)) * 91;
            nums[i] = i1 + i2;
        }
        // 2. 10进制比大小
        Arrays.sort(nums);
        // 3. 10进制 转 91进制
        for (int i = 0; i < n; i++) {
            String s1 = String.valueOf(nums[i] / 91);
            // int转char
            String s2 = String.valueOf((char) (nums[i] % 91 + '0'));
            s[i] = s2 + s1;
        }
        return s;
    }

    public static void main(String[] args) {
        SortString sortString = new SortString();
        String[] s = {"B3", "D2", "F1", "A9", "D12", "A2", "C1", "Z0", "B1"};
        sortString.sort(s);
        System.out.println(Arrays.toString(s)); // ["Z0","B1","C1","F1","A2","D2","B3","A9","D12"]
    }

}
```
