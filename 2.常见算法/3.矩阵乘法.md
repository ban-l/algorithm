# 矩阵乘法

0次幂就是单位矩阵，m次幂，每次乘以原矩阵（m-1次）。

矩阵乘法需要三个矩阵

1. **原始矩阵**
2. **结果矩阵**
3. **记录矩阵**（临时矩阵，存储矩阵的中间状态，计算后需要初始化为0）

```java
int[][] original = new int[n][n];
int[][] result = new int[n][n];
int[][] temporary = new int[n][n];
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        int k = 0;
        while (k < n) {
            // 阶乘计算，temporary记录中间状态
            temporary[i][j] = result[i][k] * original[k][j];
            k++;
        }
        result = temporary;
        temporary = null;
    }
}
```
