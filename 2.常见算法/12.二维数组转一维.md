# 二维数组转一维

## 示例：扫雷

2D 游戏的地图可以抽象成一个二维矩阵，以扫雷举例，我们可以用下面这个类表示扫雷的棋盘。

```java
public class Game {
    
    int m;
    int n;
    // 大小为 m x n 的二维数组，坐标(i,j)值为 true 表示有雷，false 表示没有雷
    boolean[][] board;
    
    public Game(int m, int n) {
        this.m = m;
        this.n = n;
        this.board = new boolean[m][n];
    }
    
    /**
     * 将二维数组中的坐标(i,j) 转换成一维数组中的索引
     */
    int encode(int i, int j) {
        return i * n + j;
    }
    
    /**
     * 将一维数组中的索引转换成二维数组中的坐标(i,j)
     */
    int[] decode(int index) {
        int i = index / n;
        int j = index % n;
        return new int[]{i, j};
    }
    
    public static void main(String[] args) {
        Game game = new Game(6, 6);
        System.out.println(game.encode(2, 3)); // 15
        System.out.println(Arrays.toString(game.decode(game.encode(2, 3)))); // [2, 3]
    }
    
}
```
